Dataverse CRUD Opertion samples

This Repository contains sample code on how to perform CREATE,READ, UPDATE, DELETE, RETRIEVE and RETRIEVE MULTIPLE Operations in Microsoft Dataverse using the .NET SDK

These samples are intended to help techical developers to understand on how to connect o Dataverse/MSCRM and perform CRUD Operaions

What he have done here

1. Connect using ServiceClient
2. Create Records in Dataverse
3. Retrive Records with RETERIVE Operations
4. Retrive Multiple Records using RETRIEVE MULRIPLE Operations
5. Update Recrods using ID
6. Delete Recrods using ID

Technology Used

1. Simple Console application with .Net Framework
2. Microsoft Dataverse SDK
3. Microsoft.PowerPlatform.Dataverse.Client (Get from Manage Nuget Packages)
4. Visual Studio 2022.
